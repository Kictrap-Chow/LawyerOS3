# LawyerOS 使用指南

## 🚀 如何在其他电脑上运行

由于您通过坚果云同步了整个文件夹，在其他电脑上运行非常简单。

### 步骤 1：安装 Node.js (只需做一次)
软件运行需要 Node.js 环境。
- **Windows**: 访问 [nodejs.org](https://nodejs.org/) 下载并安装 (推荐 LTS 版本)。
- **Mac**: 同样访问官网下载安装，或使用 `brew install node`。

### 步骤 2：启动软件
进入 `Nutstore Files/Case Management/LawyerOS2` 文件夹：

- **Windows 用户**: 双击 `start.bat` 文件。
- **Mac 用户**: 
  1. 直接双击 `start.command` 文件。
  2. **如果提示"无法执行"或"没有权限"** (常见于云同步后的新电脑)：
     - 打开终端 (按 Command+空格，搜索 Terminal 或 终端)。
     - 输入 `chmod +x ` (注意最后有个空格)。
     - 将 `start.command` 文件拖入终端窗口。
     - 按回车键。
     - 以后就可以直接双击运行了。
  3. 如果提示"无法打开，因为来自不明身份开发者"：
     - 在 Finder 中右键点击 `start.command`
     - 选择"打开"
     - 在弹出的窗口中点击"打开"即可

软件会自动下载所需的依赖（第一次运行时）并启动浏览器。

### 步骤 3：同步数据
软件启动后：
1. 点击左下角的 **Link Local File** 按钮。
2. 选择同目录下的 `database.json` 文件。
3. 这样您的数据就会通过坚果云自动同步了。

---

## ⚠️ 注意事项
1. **保持连接**：每次重新打开软件，都需要重新点击 **Link Local File** 来连接数据文件（这是浏览器的安全机制）。
2. **不要改名**：请勿修改 `database.json` 的文件名，否则多台电脑可能无法识别同一个数据库。
